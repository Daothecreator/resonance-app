# Resonance

Free therapeutic frequency PWA. No registration, no data collection, no payments.

**Live:** https://resonance-app.pages.dev

## Features
- 8 precision frequency modes (Tibetan Bowl, Bell Resonance, Golden Ratio, 528 Hz, 432 Hz, 40 Hz Gamma, Binaural Beats, Research Zone)
- 5 languages: English, Русский, Українська, العربية, 中文
- Web Audio API — pure sine wave generation with exact harmonics
- Canvas particle visualization reacting to audio
- Glassmorphism UI with dark theme
- PWA — works offline
- Timer with presets
- No ads, no tracking, no accounts

## Disclaimer
This application is for educational and relaxation purposes only. It is not a medical device.
